Operating Systems Architecture

Process Scheduling

Analyze performance metrics across Preemptive and Non-Preemptive schedulers like FCFS, Round Robin, and SRTF.

Deadlocks

Study Mutual Exclusion conditions, Wait-For Graphs, and preventive strategies using the Banker's Algorithm.

Memory Allocation

Examine dynamic storage challenges, internal/external fragmentation, and placement metrics like First-Fit and Best-Fit.

Virtual Paging

Understand Translation Lookaside Buffers (TLB), page faults, and cache eviction logic like LRU, FIFO, and Optimal Page Replacement.

Synchronization

Solve classical concurrency bugs using Semaphores, Mutex locks, Monitors, and structural Critical Section rules.

Disk Management

Optimize I/O seek times using disk-arm track movement algorithms like SSTF, SCAN, C-SCAN, and LOOK.

Analyze operational boundaries of Symmetric/Asymmetric cryptosystems, Firewalls, NAT traversal, and IPSec configurations.
